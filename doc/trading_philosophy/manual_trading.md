# 散户手动交易哲学：强趋势共振的Beta策略

> **核心洞见**
> 1. **强趋势共振**是唯一一个散户手动交易、无需回测、就很有可能具备优势的策略——这是 **Beta 策略**。
> 2. **盘感只是锦上添花**, 真正起作用的是期货的市场特性和强趋势共振策略本身的优势。
> 3. **期货是理想赛道**：往往多头或空头都有强趋势行情, 趋势出现频繁、延续时间长、波动大、自带杠杆。

---

## 1. 核心理念：Beta 策略 vs Alpha 策略

### 1.1 两种策略的本质区别

| 维度 | Alpha 策略 | Beta 策略 |
|------|------------|-----------|
| **优势来源** | 个人能力、特殊模式 | 市场内在结构、不可抗力 |
| **回测依赖** | 必须回测证明有效 | **无需回测**, 市场特性存在即有效 |
| **可复制性** | 低（因人而异） | 高（规则可量化） |
| **衰减风险** | 高（被市场学习后失效） | 低（结构性优势难以消失） |

**核心公式**：
> **强趋势共振策略（产生 Beta） + 盘感（锦上添花） = 长期优势**

### 1.2 为什么强趋势共振是 Beta 策略？

强趋势共振往往代表着**宏观的不可抗力**：
- 经济周期的转折
- 供需关系的根本性失衡
- 政策冲击的持续影响

这些宏观力量一旦启动, 就会在多个周期、多个维度上形成**共振信号**。顺应这种结构性力量, 就是在利用市场的**内在规律**, 而非试图预测或战胜市场。

**趋势跟踪从不预测, 它只是在正确的时候站在正确的一边。**

### 1.3 为什么无需回测？

- **Alpha 策略**：试图从市场中"抠出"超额收益, 必须证明模式有效, 必须回测验证
- **Beta 策略**：顺应市场内在规律, 只要市场特性存在, 策略就有效

强趋势共振的有效性不依赖于历史统计, 而依赖于市场结构本身。

❌ 不是“这个系统在任何参数下都有效”
✅ 而是“只要期货市场仍然具备趋势延续性，这类行为在统计上就不会是负期望”

只要期货市场继续存在以下特性, 这个策略就持续有效：
1. 趋势出现频繁
2. 趋势延续时间长
3. 波动幅度大
4. 自带杠杆

### 1.4 为什么几乎没有反例？

> **"至少目前, 我找不到其他策略散户手动交易能长期生存的反例。"**

**没有反例：**
*   **拼速度**：机器主场, 散户手动打不过。
*   **拼天赋**：幸存者偏差, 无法复制。
*   **拼消耗**：无法定义“垃圾时间”, 被人性耗死。

## 2. 赛道选择：为什么期货是理想赛道

### 2.1 各市场特性对比

| 市场 | 趋势特性 | 问题 | 最佳策略 |
|------|----------|------|----------|
| **加密货币** | 噪声大 | 叙事逻辑不可靠, 波动无规律 | 量化交易 |
| **外汇** | 趋势易中断 | 多国政策相互对冲 | 量化交易 |
| **股票** | 趋势慢且短 | 涨跌限制, 无杠杆 | 长期持有慢慢复利 |
| **期货** | 趋势频繁、延续长、波动大、自带杠杆 | — | **Beta 策略手动交易** |

### 2.2 期货市场的结构性优势

期货市场为什么最适合散户用 Beta 策略咸鱼翻身？

1. **趋势延续性强**：商品价格受供需驱动, 一旦供需失衡开始, 往往会持续相当长的时间
2. **杠杆自带**：无需借贷即可获得杠杆效应, 放大趋势收益
3. **多空双向强趋势**：往往多头或空头都有强趋势行情, 不怕熊市, 机会加倍
4. **流动性好**：主力合约进出顺畅, 滑点可控

**结论**：期货是最适合散户用 Beta 策略手动交易的市场, 这就是理想赛道。

## 3. 强趋势共振策略：核心定义

### 3.1 策略核心

**强趋势共振策略**意味着：
- **多周期趋势共振**：从周线到分钟线, 趋势方向一致
- **顺势而为**：只做趋势方向的单子
- **永不逆风做单**：绝不试图逆势博取反转

### 3.2 共振条件（指数+品种都必须满足）

*(注：期货市场多空皆可做。下文为了描述方便, 统一以**做多**为例, 做空逻辑反之亦然。)*

> **不满足共振条件 = 垃圾时间, 无需看盘**

#### 方案一：经典四周期共振

| 周期 | 共振条件 ( `x>` 上穿, `>` 上方 ) |
|------|----------|
| **5分钟** | close `x>` EMA `OR` 开盘第一根阳线: close `>` EMA |
| **1小时** | (Diff - Dea) `>` 0 (红柱) `AND` close `>` EMA |
| **日线** | CCI `>` 30 `AND` close `>` EMA |
| **周线** | CCI `>` 80 `AND` close `>` EMA |

> **做空逻辑对称**：做空时条件与做多完全对称：
> - 5分钟：close `x<` EMA (下穿) `OR` 开盘第一根阴线: close `<` EMA
> - 1小时：(Diff - Dea) `<` 0 (绿柱) `AND` close `<` EMA
> - 日线：CCI `<` -30 `AND` close `<` EMA
> - 周线：CCI `<` -80 `AND` close `<` EMA

> **澄清**：上述所有 `close` 均指**该K线已走完后的收盘价**, 而非盘中动态变化的最新价。

**特点**：最严格, 信号最少, 但胜率和盈亏比最高。周线 CCI > 80 确保宏观趋势已确立, 日线 CCI > 30 则进一步放宽以尽可能捕捉趋势延续中的所有机会——既然周线已经够强, 日线只需确保在常态区间上方即可。**在本体系中, 任何提高交易频率的调整, 都必须以不削弱周线级趋势许可为前提。**

> **为什么要严格区分 `x>` (上穿) 和 `>` (上方)？**
> *   **大周期 (小时/日/周) 是看方向的 (`>`)**：只要维持在上方, 说明大环境是顺风的, 就是安全的共振区域。
> *   **小周期 (5分钟) 是找买卖点的 (`x>`)**：必须精确锁定**启动的那一刻**。
>     *   如果 5分钟也是只要 `>` (在上方) 就可以交易, 就会失去节奏感, 变成随意的追涨杀跌, 导致交易混乱。
>     *   **`x>` 代表了有约束、有节奏的交易。** 只在刚刚启动的一瞬间入场, 过时不候。

> **辅助参考：周线 ADX (趋势强度)**
> *   **观察**：如果 **周线 ADX < 25**, 说明大周期趋势可能偏弱或进入震荡。
> *   **对策**：**不改变当前所有进场规则和离场规则**。只是下调预期，建议 **5分钟 1:2 直接离场**，别拿太久。
> *   **逻辑**：周线 ADX 不是用来判断“要不要做”，而是用来判断“值不值得久拿”。当 ADX 偏低时，趋势仍然可能成立，但 Beta 回报密度下降。此时应主动降低预期，避免把短线行情强行当成中长趋势来持有。
> *   **注意**：ADX 只是参考提示，不影响共振和买点判断。在程序算法中，**ADX 不参与共振信号的生成逻辑**（不会因为 ADX 低而过滤掉信号），也不影响方向判断。它仅仅计算并展示出来，供交易员人工决策"持有预期"时参考。

#### 方案二：日内偏好四周期共振

> **逻辑条件与方案一完全一致, 仅将周期整体下移一级**：
- 周期改成: **3分钟** **15分钟** **1小时** **日线**

**特点**：信号稍多, 适合日内+隔日操作。**如果实践中发现趋势不够强, 震荡多, 还是回到方案一更可靠。**

#### 方案三：跨周期 MACD 简化版

- 在 **5分钟** 图上, 使用 **4 个跨周期 MACD 指标**（对应不同周期）
- 当 4 个 MACD 同时红柱时, 形成多周期共振
- 在 5 分钟顺势而为短线操作, 目标 **1:2 盈亏比**

**特点**：操作最方便, 一屏显示所有信息, 但可能失去一些宏观视角。

### 3.3 共振强度分级

*（所有等级都必须满足品种自身的四周期共振：周线 → 日线 → 小时 → 分钟）*

*   **最优解 (5星)**：品种(四周期共振) + 板块/大盘(四周期共振) = **可做** (系统性行情)
*   **次优解 (4星)**：品种(四周期共振) = **可做** (结构性行情, 手动确认大盘环境)
*   **禁止 (垃圾时间)**：品种未形成四周期共振 = **坚决不做**

> **核心逻辑**：
> *   **最优解** = 品种+板块/大盘共振, 完美的 Beta 红利。
> *   **次优解** = 品种自身四周期共振已成立, 板块/大盘由人工确认。
> *   **禁止** = 品种自身都没共振, 没有入场理由。**宁可错过, 绝不逆势。**

> **为什么4星只看品种？**
> *   增加交易机会——不同板块的周期不同步, 总有品种在走趋势。
> *   某些品种会特立独行——即使板块/大盘横盘, 个别品种可能因自身供需矛盾走出独立行情。
> *   板块/大盘环境由人工确认, 扫描器只负责筛选出"品种自身已共振"的机会。

> **为什么统一从周线开始？**
> *   既然4星只看品种自身, 缺少板块/大盘的保护, 就必须用更高级别的共振来弥补。
> *   周线共振意味着趋势已脱离常态区间, 宏观方向明确, 比日线更可靠。
> *   宁可信号少一些, 也要确保每次入场都有足够的"安全边际"。

## 4. 盘感的定位：锦上添花, 而非核心

### 4.1 盘感的真正作用

盘感在趋势跟随中的作用是**优化器**, 而非**发动机**：

| 盘感能做的 | 盘感不能做的 |
|------------|--------------|
| 动态自适应把握趋势节奏 | 凭空创造优势 |
| 感知趋势情绪的微妙变化 | 战胜市场结构 |
| 过滤异常信号和陷阱 | 预测未来方向 |
| 让盈亏曲线更平滑 | 替代系统性策略 |

> **关键定位**：
> *   **盘感在强趋势中是辅助**：帮助优化进场点, 顺水推舟, 锦上添花。
> *   **盘感在震荡期中是毒药**：试图在无序波动中寻找规律, 只会带来无尽的亏损和心态崩溃。

### 4.2 进场环节的盘感：一票否决

**权限定义**：在进场环节，盘感只有**减法权**（不做），没有**加法权**（强做），更没有**修改权**（早做/晚做/强行做/反复做/反着做）。

1.  **不可干扰方向和节奏**：
    *   交易的方向和进场节奏，完全由策略规则死板定义 (扫描器程序化通知)。
    *   盘感**绝不可干扰**或试图修改这些硬性信号。

2.  **仅有的否决权**：
    *   当扫描器发出信号时，若盘感强烈觉得"不对劲" (如直觉到重大阻力位、形态震荡丑陋、极端暴涨暴跌)，**可以选择不做 (放弃该信号)**。
    *   **禁忌**：但**绝不能反着做**，也绝不能在无信号时**强行做**，或者被打止损后**反复做**。

3.  **禁止随意回补**：
    *   如果被打止损，**绝不能**因为"觉得还能涨"就随意再进场。
    *   必须耐心等待扫描器发出**下一次明确信号**。只能做规则给的买卖点。

4.  **禁止加仓与反手**：
    *   **一把梭逻辑**：开仓即决战。**绝不考虑任何加仓** (无论浮盈加仓还是亏损补仓) 和 **反手** 操作。
    *   **仓位管理**：开仓前就必须决定好最终仓位。保证金占资金比例应该在 **40%-60%**。如果小于这个比例, 可以多下几手, 直到符合比例, 如果超过这个比例, 哪怕机会再好也不开仓, 风控在进场前就计算好。

5.  **止损前置思维**：
    *   **止损位置**：一般放在**最小周期 (5分钟)** 附近的**分形影线**位置。如果是**大实体K线** (如长阳线/长阴线) k线走完后的收盘价进场，止损可以放在**该K线的实体中间**位置。
    *   **风控计算**：进场前必须计算潜在亏损额。单笔止损幅度应严格控制在**本金的 2% 左右**。
    *   **放弃原则**：如果技术止损位距离过远导致潜在亏损超过 2%，**直接放弃该机会，别进场**。风控必须在进场前就计算好。

### 4.3 离场环节的盘感：由僵化到优化

> **扫描器仅提示进场**：程序只负责发现机会, 不负责提示离场。离场由人来执行。

**规则核心：先僵化 (规则), 后优化 (盘感)。**

1.  **阶段一：硬性规则 (盈亏比 < 1:2)**
    *   **禁止离场**：只要价格没有触及止损线, 也没达到 1:2 的目标位, **绝对禁止手动平仓**。
    *   **禁止主动干预**：禁止离场, 禁止反手, 禁止离场后再进场, 禁止做T, **禁止任何主动操作** (只能被动接受止盈或止损)。
    *   **逻辑**：这是最难熬的阶段, 也是最容易因为"害怕利润回撤"而瞎操作的阶段。必须用死规则锁住手脚, 给趋势呼吸的空间。要么止损离场, 要么拿到基础利润。

2.  **阶段二：盘感接管 (盈亏比 > 1:2)**
    *   当利润已有安全垫 (达到 1:2) 时, 规则退场, **盘感介入**。
    *   **选择权**：
        *   **落袋为安**：盘感觉得行情乏力, 可以直接止盈。
        *   **博取长线**：盘感觉得趋势正猛 (如周线级别大爆发), 可以继续持有。
        *   **保本护航**：将止损移动到开仓价 (保本), 然后让利润奔跑, 追求 1:5 甚至更高。

### 4.4 心理防线：盘感的越权与归因

**常见陷阱**：
在趋势末期转入震荡时，往往会出现回撤。此时最容易产生的心理问题是**赋予盘感错误的"修正权"**——试图凭直觉频繁操作以挽回损失。

**核心铁律**：
*   **盘感唯一能做的只是否定（不操作）**。
*   **严禁偷偷扩权**：任何时候，盘感都不能越过信号"瞎折腾"。

**深度认知**：
给盘感偷偷扩权，**不只是纪律问题，更是认知问题**。
*   你必须从灵魂深处理解：**这套系统的优势来源是“市场特征”（强趋势），而不是你的“盘感”**。
*   盘感仅仅是辅助。

**终极归因**：
如果严格执行这套策略后依然长期无法存活，结论只有一个：**这个市场本身坏了（不再具备趋势性）**。
*   **正确做法**：**切换市场**（去寻找下一个具备强趋势特征的一级市场）。
*   **错误做法**：试图用盘感和策略去战胜一个已经不适合的赛道。

**直觉的运作模式**：
判断趋势是否衰减、行情是否震荡、买卖点是否合适，**应该只是盘感的感觉**。
*   它是长期看盘后的**经验直觉**，以及整合当下多维度信息的**潜意识涌现**。
*   **拒绝逻辑化**：避免过度规则化，避免过度解读。如果试图给每一个直觉都编造一套逻辑，必然陷入混乱。

### 4.5 认知纠偏

**旧认知**：盘感是核心, 规则是护栏
**新认知**：市场特性是核心, 强趋势共振是策略, 盘感是优化器和过滤器

盘感本身只是锦上添花。真正起作用的, 还是：
1. 期货的市场特性（趋势延续性强）
2. 强趋势共振策略本身的结构性优势

## 5. 执行框架

### 5.1 垃圾时间管理

既然共振条件已经量化, 那么：
> **不满足共振条件的时间 = 垃圾时间 = 不看盘**

这让"物理冻结"从"强制不看盘"升级为"条件未触发=不看盘"——更加主动和优雅。

### 5.2 操作流程

```
1. 检查共振条件
   ├── 不满足 → 垃圾时间, 关闭软件
   └── 满足 → 进入看盘模式

2. 看盘模式
   ├── 自上而下观察（指数 → 品种）
   ├── 寻找买卖点
   └── 用盘感微调进场时机

3. 持仓管理
   ├── 顺势持有
   ├── 止损放在结构位
   └── 目标盈亏比 1:2 或更高
```

### 5.3 关键纪律

1. **永不逆风做单**：只做共振方向
2. **垃圾时间不看盘**：条件不满足就是休息时间
3. **顺势而为**：趋势是朋友, 不要试图逆势博取反转
4. **控制仓位**：期货自带杠杆, 仓位控制在 40%-60%

### 5.4 辅助工具：趋势共振扫描器

> **定位**：这是一个**单纯的行情扫描与通知工具**, 不进行自动交易, 也不进行回测。它的唯一目的是代替人工全天候盯盘, 将人从垃圾时间中解放出来。
>
> **这个是用来给手动交易用的, 虽然放在本项目, 但是和本项目的量化交易框架无关, 这个工具和本项目的其他模块保持互相独立**

**执行逻辑：**

1.  **定时扫描**：全天候定时扫描。
2.  **共振判断**：
    *   **技术标准**：严格按照 **方案一: 经典四周期共振** 计算指标和判断共振。
    *   **筛选策略**：严格按照 **3.3 共振强度分级** 进行过滤。
    *   **只通知 5星(最优解) 和 4星(次优解) 的机会**。
    *   **其他情况(垃圾时间)一律静默**, 屏蔽所有非共振噪音 (即使品种自身走强也不通知)。
3.  **结果输出**：打印日志并 **实时推送** 交易节奏 (5分钟是否上穿EMA)。

**关键设计细节补充：**

1.  **循环间隔与周期对齐**：
    *   循环间隔时间应严格等于**最小K线周期**（如配置为 5m, 则每 5分钟 循环一次）。
    *   该间隔必须**动态读取配置**, 禁止硬编码。
2.  **整点时刻触发与单次执行**：
    *   扫描动作必须卡在**整点**执行（5m的话, 例如 5:00, 5:05, 5:10...）。
    *   **严格单次触发**：每个小周期（如 5m）**只能触发一次**扫描。只有在检测到**整点切换后的第一次**数据更新时才执行扫描逻辑, 随后在当前周期内即刻锁定, 直到下一个整点周期到来。
    *   **实现技巧**：通过对比“当前获取的最新K线时间戳”与“内存中记录的上一轮已处理K线时间戳”是否不同。若不同（说明是新周期的第一次更新）, 则执行扫描并更新记录的时间戳；若相同（说明是同周期内的后续更新）, 则直接跳过, 防止重复扫描。
3.  **指标信号判定的"上一根原则"**：
    *   **规则**：严禁使用当前未收盘K线。必须锁定**上一根已完成K线 (`iloc[-2]`)** 作为判断对象, 以**上上根 (`iloc[-3]`)** 作为穿越基准。
    *   **逻辑**：
        1.  **定义匹配**：策略要求的 "close > EMA" 是指 K 线走完后的最终收盘价。若用 `iloc[-1]` 判断, 实际用的是约等于当前开盘价, **逻辑错误**。
        2.  **避免失效**：所谓"上穿"是指**两个已走完的收盘价** (`close[-3]` vs `close[-2]`) 之间的状态跃迁。若拿 `iloc[-1]` (约等于当前开盘价) 和 `iloc[-2]` (上根收盘) 比较, 它们数值几乎相等(如果不跳空的话), 既违背了上穿定义, 也算不出任何信号。
        3.  **开盘第一根**：对于"开盘第一根阳线"策略, 同样必须等这根线走完（变身 `iloc[-2]`）后才能确认它是阳线且站上均线, 开盘瞬间无法确认。
    *   **示例**：
        ```python
        # 确认"刚才那根已收盘K线"是否上穿
        is_cross_above = (close.iloc[-2] > ema.iloc[-2]) and (close.iloc[-3] <= ema.iloc[-3])
        ```
4.  **数据源限制与功能裁剪**：
    *   **现状**：目前天勤量化（TqSdk）数据源暂不支持获取实时的大盘/板块指数数据。
    *   **应对**：因此, **板块/大盘共振**功能相关字段标记为 `TODO`或`移除`, 当前版本**仅实现品种自身的共振检测**。
    *   **影响**：共振强度分级中, 原本依赖大盘共振的“5星”判定暂时无法自动输出, 均由人工后续确认, 程序只需准确输出“品种自身共振”的结果即可。

## 6. 历史演进

> 本文不是凭空出现的理论, 是过来人在长期经验中摸索出来的东西。

我的交易认知经历的迭代过程：

1. **蒙昧期**：胡乱操作, 被情绪牵着走。
2. **技术期**：迷信特殊形态与具体指标。
3. **结构期**：过度依赖历史结构, 对当下变化反应迟钝。
4. **过滤期**：试图用盘感一票否决, 但缺乏系统性。
5. **心流期**：承认盘感优势, 但忽视了现实摩擦的残酷性。
6. **污染期**：虽然意识到了持仓后的混乱（陷入盘感的无限纠偏）, 但将其错误地归结为认知偏差（变量污染）, 试图用心理学解释。
7. **平衡期**：依然把盘感当作核心, 但看穿了手续费的摩擦成本, 建立了"全流程平衡"和"每日三眼法", 从而限制频繁交易和看盘频率。
8. **共振期(最新)**：通过实战的惨痛教训, 彻底认清了 **Alpha 与 Beta** 的本质区别。
   - **共振是边界**：承认盘感无法处理无序的无方向震荡。确立了"自上而下、强共振"的硬性生存法则。
   - **跟随趋势节奏**：不再跟随市场节奏, 而是跟随趋势节奏（因为市场充满了无效波动和手动交易难以参与的信号）。
   - **Beta 优于 Alpha**：强趋势共振是 Beta 策略, 优势来自**市场特性**（期货趋势出现频繁 + 趋势延续性强 +波动性大 + 自带杠杆）, 而不是个人能力。只要市场特性存在, 策略就有效, **无需回测**。
   - **盘感降级**：盘感不再是核心, 仅用于在共振的安全区内锦上添花顺势而为。

**最终形成**：
> **强趋势共振策略（Beta 优势） + 盘感（锦上添花） + 期货赛道（市场特性） = 长期优势**
