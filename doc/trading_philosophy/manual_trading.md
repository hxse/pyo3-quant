# 盘感交易：实践出真知

> **本文核心**：知行合一。先确定边界，再建立执行系统，最后解释背后的理论。
>
> 1. **实践为本**：盘感只能从长期训练中涌现，不能从纯理论推导中诞生。
> 2. **节奏主轴**：默认只做 5m/15m 的 1:1/1:2 短线。
> 3. **拿更久定位**：理论上可行，实操不建议。它更像挑战极限，不是日常执行。
> 4. **系统涌现**：盘感 + 固定节奏 + 硬约束 + 风控，共同构成可持续系统。

---

## 1. 前言与边界

**场景定位**：面向普通散户的小资金手动交易，目标是快速积累本金。

- **默认节奏**：只做短线 1:1/1:2，不把"拿更久"纳入日常执行。
- **周期分工**：大周期（1h/1d/1w）定方向，小周期（5m/15m）找买卖点。
- **交易形态**：非高频、非剥头皮，不追求次秒级博弈。
- **信息边界**：不依赖基本面解读，只看价格行为与当下结构。

---

## 2. 执行系统（怎么做）

### 2.1 训练入口：先模拟盘，再实盘

- **零成本试错**：先在模拟盘完成密集训练。
- **资格门槛**：模拟盘长期稳定后，再进入实盘。
- **样本要求**：训练必须跨不同市场环境（趋势、震荡、极端）。

### 2.2 固定节奏：5m/15m + 1:1/1:2

- **反馈快**：小周期反馈密度高，盘感成长快。
- **效率高**：不依赖大行情，波动存在就有机会。
- **聚焦干净**：入场靠盘感，出场靠纪律，避免系统分裂。

### 2.3 硬约束：开仓即终局

- **进场前锁定**：先锁定仓位与目标盈亏比，再下单。这里的 `1:1/1:2`，通常指 `5/15m` K线分形突破结构下的盈亏比。
- **进场后不纠错**：不跟踪止损、不上移止损、不保本平仓，**禁止纠错、反手、加仓、减仓**。
- **开盘首K约束**：小周期开盘后的第一根K线，必须等走完再进场，**禁止首K未结束进场**。
- **止损锚定**：止损一般挂在附近分形低点（或高点）；若是突破长阳线（或长阴线），止损可放在该长K中位附近；若是连续阳线进场，止损可放在前一根K线低点（连续阴线同理放前一根K线高点）。
- **固定出场**：只接受 1:1/1:2 的预设结果，要么碰止盈要么碰止损。若做 `1:1`，止盈应额外让 `2` 跳，用于覆盖潜在手续费与摩擦成本。

### 2.4 扫描执行：只筛选，不解读

- **扫描态度**：几秒内看出流畅就参与，看不出来就过滤。
- **解读陷阱**：反复琢磨每个波动，只会把噪音当信号。
- **执行原则**：不预测，不解释，不恋战。

### 2.5 风险与频率

- **单笔风险**：约本金 2%。
- **名义仓位**：约本金 40-60%（通过杠杆调节）。
- **交易频率**：每天最多开仓 3 次；建议只关注 3-5 个品种。
- **连续性**：每天固定三次看盘（上午, 下午, 晚上, 每次约 30 分钟），不中断节奏。怕累的话，夜盘可以不看。
- **节奏优先于停手**：不采用“当日连亏 N 笔或亏损达 X% 就收工”的硬熔断。以平常心看待连续亏损，保持交易节奏不变，避免陷入错过行情变化的恶性循环。
- **资金安排**：一开始只用小资金，连续亏损时的绝对损失可控；后期优先用利润继续交易，减轻心理压力，保持执行稳定。

### 2.6 环境与状态

- **状态优先**：生活、情绪、专注度不稳时，宁可不做。
- **噪音隔离**：社交干扰和环境噪音会直接破坏盘感质量。
- **心流条件**：环境没有干扰，才能形成长期心流；长期心流是盘感稳定的必要条件。
- **天赋现实**：天赋有差异，但长期训练可显著弥补。

## 3. 理论依据（为什么这样做）

### 3.1 为什么短线是默认节奏

- **依赖对象不同**：短线更依赖自身盘感训练质量；拿更久更依赖市场给大行情。
- **机会密度不同**：短线依赖小周期波动，机会更密；拿更久在震荡期试错成本更高。
- **系统稳定性不同**：短线节奏单一，可复制；混入拿更久容易造成混乱，往往是自讨苦吃。

### 3.2 盘感如何运作

盘感不是玄学，而是长期训练形成的模式识别能力。其核心现象有两个：
盘感主要运行在潜意识层面；事后解释往往是干扰，不应作为决策依据。

- **内生语言**：看到行情时会出现模糊但稳定的内在提示（如"有戏"，"不对劲"，"不够流畅"，"这支撑可以"，"这超卖有戏"等）。
- **自动聚焦**：注意力会自动落在最清晰的周期与信号上，而不是理性挑选。

盘感常按两阶段运行：

1. **宏观扫视（行情分类）**：先把行情分为垃圾/流畅/极端。
2. **聚焦定位（执行落点）**：再在大周期定型，在 5m/15m 找具体买卖点。

### 3.3 为什么这套体系有正期望

1. **盘感灵活**：可在不同市场状态下切换聚焦。
2. **机会密度**：小周期机会更多，不必经常死等大行情，盘感有更大发挥空间。
3. **训练稳态**：长期模拟盘训练会显著提升盘感稳定性，减少执行漂移。
4. **规则极简**：减少可调参数，避免盘感被噪音污染。
5. **风险前置**：开仓前算清风险，避免事中情绪化操作。
6. **防自我毁灭**：限频、止损、仓位约束，避免常见死法。
7. **筛选而非预测**：只参与清晰结构，主动过滤垃圾行情。

### 3.4 正期望的最小数学框架

这套体系不是靠“感觉对一次”赚钱，而是靠**长期样本下的期望值**赚钱。

- **期望值公式**：`Expectancy = 胜率 * 盈利R - 败率 * 亏损R - 摩擦成本`。
- **在本体系中**：亏损R通常固定为1，盈利R通常取1或2（即1:1或1:2）。
- **含义**：
  - 做 `1:1` 时，胜率需要长期高于约 50%（且要覆盖手续费与滑点）。
  - 做 `1:2` 时，胜率门槛可降到约 33% 以上（同样要覆盖摩擦成本）。
- **结论**：核心不是追求“每一单都对”，而是让“胜率-盈亏比-摩擦成本”长期站在你这边。

### 3.5 为什么执行一致性比单笔正确更重要

- **单笔结果噪音很大**：一两笔的盈亏不能证明策略优劣。
- **一致性才有统计意义**：固定节奏、固定风控、固定出场，样本才可比较。
- **系统稳定来自可重复**：当流程一致时，盘感训练成果才能被放大；当流程频繁变动时，优势会被噪音抵消。
- **实操结论**：宁可“长期稳定地做对80分”，也不要“偶尔做出100分但不可复制”。

## 4. 边界与误区

### 4.1 关于"拿更久"

**结论**：理论承认可行，实操不建议。

拿更久只用于挑战自身极限，不用于日常执行。常规情况下，坚持短线 1:1/1:2 即可，效率已足够高。

原因很简单：

- 它依赖大行情，不是随时可用。
- 它与短线的盘感状态不同，混用会引发节奏混乱。
- 它更容易把系统从"可复制"拉向"看天吃饭"。

### 4.2 盘感 vs 量化 / 机器学习

- **量化与 ML**：在既定框架内优化。
- **盘感**：在实时交易中重定义"什么是信号"。
- **结论**：两条路都可行，但本文讨论的是手动盘感路径。

### 4.3 为什么不做剥头皮/高频

- 高频是机构级基础设施博弈，个人没有比较优势。
- 剥头皮频率过高，身心消耗不可持续。
- 5m/15m 的 1:1/1:2 在反馈速度、可执行性与可持续性之间更平衡。

### 4.4 常见误区：把“解释能力”误当“交易能力”

- 交易后给出漂亮解释，并不等于交易前有稳定优势。
- 盘感主要在潜意识层运行，事后解释常常只是“结果合理化”。
- 真正有价值的是：是否长期遵守同一执行协议，并在足够样本下保持正期望。

## 5. 附录

### 5.1 旧版"拿更久上策"为何被修正

旧版并非纯错，而是前提不同：它假设盘感不可靠，所以主张减少决策次数。新版把"可训练、可提升"作为前提，且更重视震荡期的真实摩擦成本，因此将短线确立为默认节奏。

### 5.2 认知演进（摘要）

1. 从情绪化乱做，到迷信技术形态。
2. 从结构迷信，到尝试过滤。
3. 从空谈盘感，到回归实践训练。
4. 最终形成：固定节奏、短线主轴、筛选执行。

> **实践出真知，知行合一。**
> **保持训练节奏的长期稳定性。**
> **默认只做短线 1:1/1:2。**
> **理论承认拿更久可行，但实操不建议。**
> **不理解、不解读、不预测、只筛选。**
